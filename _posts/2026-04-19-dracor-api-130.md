In the wake of the latest release of
[DraCor Schema release](https://blog.dracor.org/posts/dracor-schema-140/)
we also updated several other components of the DraCor ecosystem:

- the [DraCor API 1.3.0](https://github.com/dracor-org/dracor-api/releases/tag/1.3.0)
  supports the new markup introduced with DraCor Schema 1.4.0 while remaining
  compatible with previous schema versions.
- the [DraCor Validate Action 2.4.0](https://github.com/dracor-org/dracor-validate-action/releases/tag/2.4.0)
  has been updated to the latest DraCor Schema and TEI versions.
- the
  [first official release of the DraCor Oxygen Framework](https://github.com/dracor-org/dracor-oxygen-framework/releases/tag/1.0.0) makes it easier to validate DraCor
  documents in the [Oxygen XML Editor](https://www.oxygenxml.com) and to stay up
  to date with DraCor Schema releases.
